现代算法

模拟退火

特别针对TSP问题的启发式算法。

步骤

  1. 初始化：初始温度T（足够大）、初始解状态s（算法迭代起点）、每个T值的迭代次数L;
  2. 寻找解：对k=1:L：
    • 产生新的解
    • 计算新增能量（f为目标评价函数）
    • ==若否则以概率接受s=t==
    • 达到终止条件，输出最优解（一般终止条件为没有新解被接受）
  3. 降温：根据T，返回2迭代

收敛到最优解的一般条件

  1. T足够高
  2. 热平衡时间足够长
  3. 终止时间足够长
  4. 下降足够缓慢

一般产生新解方法

倒序法

在2-101之间产生两个随机数u，v之间的数全部倒序排列（包括u，v），产生新的解。

变异法

在2-101之间产生三个随机数u，v，w将u，v之间的数全部移动到w之后，产生新的解。

实例及算法实现

遗传算法

基于自然选择和自然遗传机制的搜索寻优算法。

思路

  1. 确定可行域和量化方式
  2. 对每个解构造衡量解的优良性的适应度函数
  3. 确定进化参数群体规模、交叉概率、变异概率、进化终止条件

步骤

  1. 设置初始参数：种群大小，最大代数，交叉概率、变异概率；
  2. 可行解编码；
  3. 确定初始种群；
  4. 交叉操作；
  5. 变异操作；
  6. 选择适应度高的父代和子代构成下一代；
  7. 迭代。

基于高响应比值优先动态调度算法（HRRN）和贪婪算法（Greedy）的动态加工安排

优先原则：

2. Greey: f值相等，以t排序
3. 随机: f,t相等，随机排序
