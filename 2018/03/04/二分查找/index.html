二分查找

2018-03-04

概念

在数组中找到对应数据的下标的一种方法(优于遍历)。
要求：数组有序。

思路

利用数组中间值与查找数据比较，确定查找数据位置。
因而其时间复杂度为O(log2n)

代码

public int binarySearch(int value, int[] array){
    int lo = 0;
    int up = array.length - 1;
    while(lo <= up) { // 循环比较至查找结束
        int key = (lo + up) / 2; // 取中间值比较
        if(value > array[key]) {
            lo = key; // 改变下界 缩小范围
        }
        else if(value < array[key]) {
            up = key; // 改变上界 缩小范围
        }
        else {
            return key; // 查找到key
        }
    }
    return -1; // 未查找到
}
