FibonacciSequence简介

FibonacciSequence即斐波那契数列，指的是由 A1 = 1,A2 = 1,且 An+1 = An + An-1,组成的数列。
例：1，1，2，3，5，8···

相关算法

以下代码仅写出关键模块

1.for循环实现 - 初级版

int fi1 = 1;
int fi2 = 2;
if (input <= 2) {
    switch (input) {
        case 1:
            System.out.print("1");
            case 2:
            System.out.print("1 ,");
            System.out.print("1");
        }
    }
else {
    System.out.print("1 ," );
    System.out.print("1 ,");
    for (int i = 0;i < input - 2 ;i++ ) {
        if (i % 2 == 0) {
            fi1 = fi1 + fi2;
            System.out.print(fi1 + " ,");
        }
        else {
            fi2 = fi1 + fi2;
            System.out.print(fi2 + " ,");
        }
    }
// 用两个变量交替相加实现

2.for循环 - 进阶版

int f = 0;
int g = 1;
for (int i = 0;i <= input ;i++ ) {
    System.out.println(f);
    f = f + g;
    g = f - g;
}
// 原理同上，写法更加简洁

3.递归实现

public static int F(int i){
    if (i == 1) {
        return 1;
    }
    else if (i == 2) {
        return 1;
    }
    return F(i - 2) + F(i - 1);
}
// 简便代码，更易读
