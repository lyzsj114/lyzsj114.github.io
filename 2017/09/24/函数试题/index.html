试题

编程/数学挑战题
函数f(n)满足以下条件：
1）定义域和值域都是正整数；
2）f(n) < f(n + 1);
3）f(f(n)) = 3n;
求 f(2017).

思考

∵ 当 f(1) = 1 时，明显不成立；
当 f(1) = 2 时，成立；
当 f(1) >= 3 时，均不成立；

∴ f(1) = 2;
f(2) = 3;
······

分析可知：
两个被3整除的函数值间，函数值均匀分布。

代码

class Test{
    public static void main(String args[]){
        System.out.println(f(2017));
    }
    public static int f(int n){
        if (n == 1) {
            return 2;
        }
        else if (n == 2) {
            return 3;
        }
        else if (n % 3 == 0) {
            return 3 * f(n / 3);
        }
        else {
            int nm = n;
            int na = n;
            // 取得小于n且能被3整除的数
            while (nm % 3 != 0) {
                nm--;
            }
            // 取得大于n且能被3整除的数
            while (na % 3 != 0) {
                na++;
            }
            return f(nm) + (f(na) - f(nm)) / (na - nm) * (n - nm);
        }
    }
}
//3864
//：~

代码分析

1. 采用递归函数形式求解，省去繁琐循环；
2. 代码简介易读。
