Java错题

2017-07-29

题目

运行下面程序，结果为：

public class Demo {
    public static void main(String args[]) {
        int num = 50 ;
        num = num ++ * 2 ;
        System.out.println(num) ;
    }
}

易错点

误将 num * 2 = 100 后再进行 num = num + 1
输出 101

改正

public class Demo {
    public static void main(String args[]) {
        int num = 50 ;
        num = num ++ * 2 ;
        /**
        上式分解为
        1. num * 2 = 100
        2. num = num + 1 → num = 51
        3. num = 100**/
        System.out.println(num) ;
    }
}
